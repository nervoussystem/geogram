Detailed Description

A piece of a mesh.

Stores a list of facet indices. The mesh it belongs to is supposed to have an Attribute<index_t> attached to the facets and indicating the id of the chart for each facet.

vector<index_t> GEO::Chart::facets

The list of facet indices of this chart.

The mesh is supposed to have an Attribute<index_t> named "chart" attached to the facets, and the list of facets has all the facets f for which chart[f] == id.
